# Knowledge Base

This is the curated, file-based knowledge base for the Salesforce Agentforce project. It is intentionally Markdown-only and Git-backed.

It is not a vector database and does not perform automatic semantic retrieval. Codex should read the relevant files directly before planning.

## Files

- `salesforce-agentforce.md`: Agentforce concepts used by this project.
- `project-map.md`: repository and metadata map.
- `development-workflow.md`: Codex execution workflow for requirements.
- `validation-and-release.md`: local validation, org validation, deploy, and rollback guidance.
- `knowledge-update-policy.md`: rules for updating this knowledge base.
